# TeknoGraf - Haber API Paneli

TeknoGraf, HTML, CSS, JavaScript, jQuery ve Bootstrap kullanılarak geliştirilmiş, mobil uyumlu (responsive) bir teknoloji haberleri toplayıcısıdır. Proje gereksinimlerine tam uyumlu olarak tasarlanmıştır.

## Özellikler

* Çok Katmanlı Veri Çekme: 8 farklı yerel teknoloji sitesinden (ShiftDelete, Webtekno, vb.) RSS ile veri çeker. Yedek olarak GNews API kullanır.
* Performans: Çekilen haberler tarayıcı belleğinde (localStorage) önbelleklenir ve "Daha Fazla Yükle" mantığı ile sayfa sayfa yüklenir.
* Kullanıcı Deneyimi: Gece/Gündüz modu, kategori ve kelime bazlı dinamik arama, haberleri yerel olarak kaydetme (favoriler) işlevleri mevcuttur.

## Kullanılan Teknolojiler

* HTML5 & CSS3
* JavaScript (ES6)
* jQuery
* Bootstrap

## Kurulum ve Çalıştırma

Proje yerel bir sunucu veya Node.js kurulumu gerektirmez.

1. Projeyi bilgisayarınıza klonlayın veya indirin.
2. Klasör içindeki `index.html` dosyasına çift tıklayarak varsayılan tarayıcınızda (Chrome, Edge vb.) açın.
